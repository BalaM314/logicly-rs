# logicly-rs

A parser for [logicly](https://logic.ly/) files.

## Feature support
* Components
  * IO
    * [x] Switch, Button
    * [ ] Clock
    * [x] High constant, Low constant,
    * [x] Light bulb
    * [~] 4-bit digit
  * Gates
    * [x] Buffer
    * [x] Not
    * [x] Or
    * [x] Nor
    * [x] And
    * [x] Nand
    * [x] Xor
    * [x] Xnor
    * [ ] Tri-state
  * Misc
    * [ ] Flip-flops
    * [ ] Bus
    * [ ] Pull-up
    * [ ] Pull-down
    * [x] Label
    * [ ] Integrated circuits
* Features
  * [x] Reading .logicly files
  * [ ] Writing to .logicly files
  * [x] Generating truth tables
